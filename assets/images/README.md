# Images Directory

This directory contains image assets for DuckBot Go.

## Planned Images

- **App branding:** Logo variations, splash screen graphics
- **Icons:** Custom action icons, status indicators
- **Illustrations:** Empty states, onboarding graphics
- **Avatars:** User/AI avatar placeholders

## Format Guidelines

- Use PNG for transparency needs
- Use WebP for better compression (Flutter supports it)
- Provide @2x and @3x variants for crisp rendering
- Keep file sizes optimized

## Adding Images

1. Add image files to this directory
2. Run `flutter clean && flutter pub get` to rebuild
3. Reference in code: `Image.asset('assets/images/filename.png')`
